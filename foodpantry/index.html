Cache Valley Food Pantry

This is the first website I ever built for real (real meaning I got paid). Up to this time, I had only been experimenting and playing with websites for fun. Never had I built one from the ground up with a deadline and potentially more work if I did well. I spent a lot time fixing small problems I had never experienced before.

Complete Date: October 2013
See it Live

I learned a TON about:

  • Laying out Divs - Divs on top of divs on top of divs... I was a pro at general layout code by the end of this website.
  • Media Queries - I hadn't built a fully functional website with media queries up until this point.
  • Mobile - This was the first time I build a fully responsive website.

This was definitely a challenging website to build, but it was those challenges that pushed me above the limit and forced me to become a much better developer.
