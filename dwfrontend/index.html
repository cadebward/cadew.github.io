DW Front End Solutions

DW Front End Solutions is a site that I built for a class as a final project. I built this site in conjunction with another student, Sean Duncan. We used Github to collaborate together. This was the first time I actually collaborated with another person on an active project over Github.

Complete Date: December 2013
See it live

I was able to learn a few things from this project:

  • Collaboration over Github - I was forced to learn how to solve conflicts, create meaningful commits, and pull changes made from another person. We chose to use Github just so we could learn to use it better.
  • Maintaining Code - I was able to see how easily code can become cluttered if time is not spent to keep things organized. Cluttered code is no fun!

I enjoy collaborating and coding with other people. This project reaffirmed that enjoyment for me. Who likes to code alone, anyways?
